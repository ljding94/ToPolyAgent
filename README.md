# ToPolyAgent: Multi-Agent System for Coarse-Grained Polymer Simulations

This project builds a multi-agent system using CrewAI to automate coarse-grained polymer simulations with LAMMPS.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Install LAMMPS and Packmol (via conda or manual):
   ```bash
   conda install -c conda-forge lammps packmol
   ```
